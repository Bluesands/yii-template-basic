### 结构说明
 + common 自定义第三方服务功能模块
 + components 自定义yii组件
 + config 项目配置文件
 + controllers 控制器
    - base 基础控制器
    - adm 管理后台
    - wxmini 微信小程序
 + enums 项目状态管理
 + helpers 帮助类集合
 + models 模型
    - common 基础模型核心类
    - base 分组模型基类
    - adm 管理后台
    - wxmini 微信小程序
 + services 模型与控制器中间业务处理模块
 + views 页面渲染模块
 + web 项目入口
